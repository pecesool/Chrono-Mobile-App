# Chrono - AI-усиленный календарь для студентов

## Описание
Chrono — это умный календарь, который не просто хранит события, а активно помогает студенту распределить задачи с учётом приоритетов, реалистичных оценок времени и личных паттернов продуктивности.

## Ключевые принципы
- **AI через UI, не через чат** — умные дефолты, которые пользователь видит и легко меняет
- **Rule-based AI** — не требует ML-моделей, работает сразу после onboarding
- **Визуальные Suggestion Cards** — результат AI показывается в красивых карточках с кнопками действий

## Архитектура
```
lib/
├── core/           # Темы, константы, утилиты, сервисы
├── data/           # Модели (Hive), репозитории
├── domain/         # AI-движок (scheduler, voice parser)
├── providers/      # Riverpod провайдеры
└── presentation/   # UI экраны
    ├── onboarding/     # 5-шаговый опросник
    ├── dashboard/      # Главный экран с таймлайном
    ├── task_input/     # Форма добавления задачи
    └── ai_suggestions/ # Suggestion Cards
```

## Зависимости
- `flutter_riverpod` — state management
- `hive` + `hive_flutter` — локальное хранилище
- `flutter_animate` — анимации
- `google_fonts` — шрифты
- `speech_to_text` — голосовой ввод
- `intl` — форматирование дат
- `uuid` — генерация ID

## Установка и запуск

### 1. Создай Flutter проект
```bash
flutter create chrono_app
cd chrono_app
```

### 2. Замени файлы
Скопируй все файлы из папки `lib/` этого проекта в `lib/` твоего Flutter проекта.

### 3. Добавь зависимости
Замени `pubspec.yaml` на наш и запусти:
```bash
flutter pub get
```

### 4. Сгенерируй Hive адаптеры
```bash
dart run build_runner build
```

### 5. Запусти
```bash
flutter run
```
